Linux backup on the fly

iaroki

 Fri 25 May 2012
Category: Linux
Tags: linux backup

Backing-up:

cd /
tar cvpjf backup.tar.bz2 --exclude=/proc 
                         --exclude=/lost+found 
                         --exclude=/backup.tar.bz2 
                         --exclude=/mnt 
                         --exclude=/sys /

Restoring:

tar xvpfj backup.tar.bz2 -C /
